# Data

The files in this directory give the [Indices of Muliple Deprivation 2015](https://www.gov.uk/government/statistics/english-indices-of-deprivation-2015) (Ministry of Housing, Communities & Local Government) decile for each postcode. They were created by using the `LSOA11` column in the [ONS postcode lookup](http://geoportal.statistics.gov.uk/items/national-statistics-postcode-lookup-august-2018).
